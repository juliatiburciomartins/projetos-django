
# Projeto Django

#### Este projeto foi desenvolvido com o objetivo de demonstrar, de forma simples e prática, como funciona o sistema de rotas no Django. A aplicação define múltiplas URLs e associa cada uma delas a uma função (view), retornando respostas diretas ao usuário.

## 📋 Pré-requisitos
#### Antes de executar o projeto, você precisa ter instalado em sua máquina:

- 🐍 [Python](https://www.python.org/) (versão 3.8 ou superior)
- 📦 [pip](https://pip.pypa.io/) (gerenciador de pacotes do Python)
- 🧰 [Git](https://git-scm.com/) (opcional, para clonar o repositório)
- 💻 [VS Code](https://code.visualstudio.com/) ou outro editor de código


## 🔧 Instalação

#### Passo-a-passo para criar um projeto do zero
🔹 Criar projeto Django:
   django-admin startproject nome_do_projeto

🔹 Entrar na pasta do projeto:
   cd nome_do_projeto

🔹 Criar um app:
   python manage.py startapp nome_do_app


#### Passo-a-passo para executar meu projeto.
🔹 Clonar o repositório:
   git clone https://github.com/juliatiburciomartins/projetos-django

🔹 Entrar na pasta do projeto:
   cd projetos-django

🔹 Atualizar pip (opcional):
   python -m pip install --upgrade pip

🔹 Rodar servidor Django:
   python manage.py runserver

🔹 Abrir no navegador:
   http://127.0.0.1:8000/


## Tecnologias Utilizadas
- Python - Linguagem de programação
